This directory contains the PressMint/TEI linguistically annotated version of
the PressMint-XX corpus. The root file PressMint-XX.ana.xml contains the
corpus teiHeader and XIncludes of the component files, which have the form
PressMint-XX_<suffix>.ana.xml. Also included are the XML schemas of the
corpus.
